# 🕹️ MIPS Assembly - Fundamentos (PUC-SP)

Este repositório contém os exercícios iniciais de programação em linguagem Assembly (Arquitetura MIPS) desenvolvidos para a disciplina de arquitetura de computadores na PUC-SP.

### 🚀 Exercícios Inclusos

O projeto aborda a lógica de baixo nível através de três scripts principais:

* **`ex1.asm` (Operações Básicas):** Foco em movimentação de dados entre registradores e operações aritméticas fundamentais.
* **`ex2.asm` (Estruturas de Decisão):** Implementação de desvios condicionais (*branches*) para execução de lógicas simples.
* **`ex3.asm` (Laços e Repetição):** Uso de instruções de salto (`j`, `beq`, `bne`) para criação de loops e contadores.

### 🛠️ Conceitos Técnicos
* Manipulação de registradores (`$s`, `$t`, `$v`, `$a`).
* Uso de chamadas de sistema (*syscalls*) para impressão de resultados.
* Ciclo de instrução e fluxo de execução em baixo nível.

### 🏁 Como Executar

Para rodar os exercícios, utilize o simulador **MARS**:

1.  **Download:** Baixe o simulador oficial aqui: [MARS v4.5](https://github.com/dpetersanderson/MARS/releases/tag/v.4.5).
2.  **Abrir Arquivo:** No MARS, vá em `File > Open` e escolha o arquivo desejado.
3.  **Montar (Assemble):** Clique no ícone da **Engrenagem** (ou pressione F3).
4.  **Executar:** Clique no ícone de **Play** verde para rodar o código. 
    * *Dica:* Você pode ajustar a velocidade de simulação no controle deslizante superior para observar os registradores mudando em tempo real.
